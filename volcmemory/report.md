# Mem0 读写性能测试报告

## 测试概况

|指标 | Write (写入) | Read (读取) |
|------|-------------|-------------|
| 用户范围 | user-100 ~ user-200 | user-100 ~ user-200 |
| 操作次数 | 101 次 | 101 次 |
| 单次耗时 Min | 0.042s | 0.148s |
| 单次耗时 Max | 0.140s | 0.375s |
| 单次耗时 Avg | **0.048s** | **0.213s** |

## 性能对比

| 操作 | 平均耗时 | 相对比例 |
|------|---------|---------|
| Write (async add) | 0.048s | 基准 (1x) |
| Read (search) | 0.213s | **4.4x** slower |

- **写入速度**：平均48ms/请求
- **读取速度**：平均 213ms/请求，读操作比写操作慢约 **4.4 倍**

|------|-----|
| 每次查询返回记忆数 | 2 条（所有用户一致） |
| 查询内容 | "我的名字是啥?" |

## 结论

1. **写入性能稳定**：大部分请求在 42-45ms 之间，仅5 次超过 100ms
2. **读取波动较大**：耗时范围148ms-375ms，波动率约 2.5x
3. **读取慢于写入**：读取比写入慢约 4.4 倍，符合预期（搜索需遍历匹配）
4. **数据一致性**：写入101 条用户数据，读取均返回 2 条记忆，数据完整性良好
